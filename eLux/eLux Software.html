Announcement of discontinuation
UniCon has decided to discontinue the software service and support for the eLux NG software
as of December 31, 2010.
The successor software eLux RL is offered on www.myelux.com for free migration.
This requires that the client hardware is compliant (see eLux RL) and valid subscription is available.
For any questions about the discontinuation or migration please apply to sales@unicon-ka.de.

Do only use with container UC_VIA-1.0

eLux® NG is not for free! 
If you already use eLux 1.x and Scout eL, update licenses must be bought. 
You must read the migration paper: German  English

If you are not yet using any version of  eLux 1.x, you can run eLux NG and Scout Enterprise unlicensed in test mode. Test mode means: Thin clients running eLux NG can start 40 times without license. Scout Enterprise can manage up to 3 thin clients.

This version supports eLux/Scout subscription

 

If you already use eLux NG Release 1, please be aware that BaseOS 1.1-6 must be installed to enable update to BaseOS 1.2 .
